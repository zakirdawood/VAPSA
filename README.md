# VAPSA
VAPSA is a program that can calculate the perimeter and area of 2D shapes, and the volume and surface area of 3D shapes.

# 2D Shapes Supported
- Triangle
- Square
- Rectangle
- Pentagon
- Hexagon
- Heptagon
- Octagon

# 3D Shapes Supported
- Sphere
- Cylinder
- Cube
- Rectangular Prism
- Cone
- Square-based Pyramid
- Triangular Prism
